# bitgin-mock-server-staging
BITGIN API Mock Server on Staging

- Run
```
$ bgsign --help
Usage: bgsign [options] 
  Currently, the following flags can be used
  -k string
        key
  -p string
        client serve port (default "8080")
  -s string
        secret
```

- example

```
$ bgsign -p [port] -k [KEY] -s [SECRET]
```
